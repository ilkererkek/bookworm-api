Bookworm API

A Rest API with mongoose and MongoDB for the bookworm website. All of the list responses in the API support pagination and sorting by query parameters.

GET Get Authors
{{URL}}/api/v1/authors?page=1&limit=1&sort=-name

It returns all of the authors with pagination.

PARAMS
page
1

None

limit
1

None

sort
-name

None

Example Request
  • Get Authors
                                                
GET {{URL}}/api/v1/authors?page=1&limit=1&sort=-name

DELETE Delete Author
{{URL}}/api/v1/authors/{{{{AUTHOR_ID}}}}

Deletes the Author that has the given id and cascade deletes all of the books owned by the author.

Example Request
  • Delete Author
                                                
DELETE {{URL}}/api/v1/authors/{{{{AUTHOR_ID}}}}
GET Get Books of an Author
{{URL}}/api/v1/authors/{{{{AUTHOR_ID}}}}/books?page=1&limit=1&sort=-title

Returns all of the books of Author with the givenId in the url parameter with pagination and sorting available in query.
